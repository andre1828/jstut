Variaveis

Os tipos de dados que veremos em javascript são :

  • Number
  • String
  • Boolean
  • Null
  • Undefined
  • Object

Number: sempre representa um double de precisão dupla.

String: um array de caracteres, pode ser representado por aspas simples ou duplas.

Boolean: sempre possui os valores true ou false.

Null: um objeto especial que representa ausencia de valor.

Undefined: valor inicial de qualquer variavel que tenha sido declarada, porém não inicializada.

Object: uma estrutura de dados que possui informações e métodos para processamento de informações.

Atenção : Undefined é um tipo primitivo, enquanto Null é um objeto

A criação de variáveis segue a seguinte sintaxe :

var minhaVariavel = "string qualquer" ;

Objetos e prototipagem

O programador que está familiarizado com a programação orientada a objetos, deve (espera-se) está confortável com o conceito de objeto e classe. Mas e a programação orientada a protótipos? Vamos falar sobre esse estilo de programação e como ele funciona em javascript.

A programação orientada a prototipos é tambem chamada de "sem classes" e "orientada a instancias", esses nomes nos dão algumas dicas.

Nesse estilo de programação o foco está nas instancias, não existem classes, cria-se objetos como se cria variáveis.

Observe o exemplo :

          
            var pessoa = {};
            console.log(typeof pessoa); // exibe "object" no console

Nesse exemplo é criado um objeto vazio chamado "pessoa".E os atributos? Vamos criar alguns agora :

          
            var pessoa = {};
            pessoa.nome = "José";
            pessoa.sobrenome = "Sarapatiaguara";
            pessoa.idade = 40;

            console.log(pessoa.nome)  // exibe "José" no console
            console.log(pessoa)       // exibe Object {idade: 40, sobrenome: "Sarapatiaguara", nome: "José"} no console

Sem classe, adicionamos atributos ao objeto como se eles já existissem, apenas usando a sintaxe "nomeDoObjeto.novoAtributo = valor"

Arrays

É possivel criar arrays da seguinte forma

var meuArray = [1,2,3];

em javascript, um array pode conter varios valores de tipos diferentes, inclusive outros arrays

var meuArray = [1, 2, 3, ' 4 ', true, [ 5, 6 ] ];

Para acessar o valor de um index em um array : 

          
            meuArray[0];   // retorna 1
            meuArray[4];   // retorna true

Para mudar o valor de um index em um array : 

          
            meuArray[0] = 99;       // meuArray = [99, 2, 3, '4', true, [5,6]]
            meuArray[4] = false;    // meuArray = [99, 2, 3, '4', false, [5,6]]

Alguns metodos disponiveis para realizar operações em arrays são :

  • Push : Adicionar algo ao array
  • Splice : Eliminar algo de um array
  • Join : converte todos os valores para string e os combina

Array.push() recebe como argumentos o valor a ser adicionado e retorna o novo tamanho do array

          
          var arrayQualquer = ['a','b','c'];
          arrayQualquer.push('MyProject'); // arrayQualquer = ['a', 'b', 'c', "MyProject"]; e retorna 4

Array.splice() recebe como argumentos o indice e a quantidade de itens a serem removidos a partir do indice dado. E retorna um array com os itens removidos. 

          
            arrayQualquer.splice(1, 3); // arrayQualquer = ['a']
                                        // retorna ['b', 'c', "MyProject"]

Array.join() recebe como argumento um separador, converte todos os itens do array para suas versões string, e os combina em uma string final onde todos os itens são separados pelo argumento dado (valores null e undefined viram strings vazias).

meuArray.join(' + '); // retorna "1 + 2 + 3 + 4 + true"

Maps

É possivel criar maps da seguinte forma

var meuMap = new Map(); // meuMap = {}

Adicionar chave e valor :

meuMap.set("minhaChave", "meuValor") // Map {"minhaChave" => "meuValor"}

Consultar valor :

meuMap.get("minhaChave") // retorna "meuValor"

Para alterar o valor em uma chave basta a inserir com o novo valor

meuMap.set("minhaChave", "novoValor") // Map {"minhaChave" => "novoValor"}

Estruturas de repetição

Javascript sendo uma lingua da familia C (sintaxe inspirada na linguagem C), possui estruturas de repetição com sintaxe familiar.

For

percorrer um array usando for :

      
        var meuArray = [1,2,3];
        for(var i = 0; i < meuArray.length; i++) {
              console.log(meuArray[i]);
        }
        // imprime no console : 1
        //                      2
        //                      3

While

Um exemplo de uso do while : 

        
          var conta = 0;
          while(conta < 10) {
            console.log(conta);
            conta ++;
          }

Manipulando o DOM

Javascript possui uma api para criar encontrar, e manipular elementos html

Encontrando elementos :

Por id

        
          var resultado = document.getElementById('variaveis');
          // resultado se refere a 1, e somente 1 elemento html que possua um id correspondente
          // Esse código retorna o titulo "Variáveis" dessa pagina.

Por classe

        
          var resultado = document.getElementsByClassName('language-javascript');
          // 'resultado' contém uma lista com todos os elementos que possuem a classe "language-javascript"
          //  Você pode testar esse exemplo nessa pagina, ele retorna todas as seções com exemplos de código

Por tag

        
          var resultado = document.getElementsByTagName('p');
          // 'resultado' contém uma lista com todas tags <p></p>
          //  Esse resultado retorna todas as tags p dessa pagina

Criando elementos com document.createElement() :

        
            var novaDiv = document.createElement('div'); // novaDiv agora se refere a uma instancia da tag <div></div>

Aninhando elementos com .appendChild() :

      
            var novaDiv = document.createElement('div'); // novaDiv = <div></div>
            var novoP = document.createElement('p');     // novoP   = <p></p>
            novaDiv.appendChild(novoP);                  // novaDiv = <div>
                                                         //              <p><p>
                                                         //           <div>

Adicionando e removendo atributos de elementos html

Adicionando atributos :

      
          var p = document.createElement('p');            // p = <p></p>
          p.className = 'centralizar';                    // p = <p class="centralizar"></p>

Ou usando setAttribute() :

      
          var p = document.createElement('p');            // p = <p></p>
          p.setAttribute('class', 'centralizar');         // p = <p class="centralizar"></p>
      
    
      
          var p = document.createElement('p');            // p = <p></p>
          p.setAttribute('id', 'idUnico');                // p = <p id="idUnico"></p>

nem todos os atributos podem ser acessados e modificados diretamente como no primeiro exemplo "(p.className = 'centralizar')", o método setAttribute() oferece maior consistencia.

Removendo atributos :

      
          var p = document.createElement('p');            // p = <p></p>
          p.setAttribute('id', 'idUnico');                // p = <p id="idUnico"></p>
          o.removeAttribute('id')                         // p = <p></p>

Consultando atributos :

      
          var p = document.createElement('p');            // p = <p></p>
          p.setAttribute('id', 'idUnico');                // p = <p id="idUnico"></p>
          p.getAttribute('id');                           // retorna "idUnico"

Funções

A criação de uma função segue a seguinte sintaxe : 

      
          function minhaFuncao() {
            console.log("opa, chamou?");
          }

          minhaFuncao() ; //  imprime "opa, chamou?" no console

Funções de primeira ordem

Observe o exemplo abaixo

      
          function executarFuncao(funcao){
            funcao();
          }

          executarFuncao(minhaFuncao); //  imprime "opa, chamou?" no console

Esse exemplo faz uso da função chamada "minhaFuncao" que criamos no exemplo anterior. "executarFuncao" é uma função que recebe como parâmetro uma outra função, e a executa.

Isso é uma demostração do uso de funções de primeira ordem ( ou primeira classe, ou ainda alta ordem, você vai se deparar com várias nomeclaturas).

Quando nos referimos a um termo na programação como sendo um termo de primeira ordem, queremos dizer que ele pode ser passado como parâmetro para uma função, bem como ser retornado por uma função e também atribuído a uma variável.

Em javascript nós faremos uso frequente dessa propriedade, por exemplo em event listeners que veremos a seguir.

Respondendo a eventos

Para responder a cliques em botões, e a centenas de outros eventos, nós usamos os EventListeners

Encontrando um botão na pagina e adicionando um EventListener

      
          var btn = document.getElementById('btnExample');     // btn corresponde ao botão encontrado ("Mostrar Alert")
          btn.addEventListener("click", function(){            // adicionamos um EventListener que aguarda um clique com botão esquerdo do mouse
            alert("Nada pra ver aqui, pode continuar u.u");    // respondemos ao evento exibindo um aviso
          });

Vamos analisar esse exemplo linha por linha

Na linha numero 2 encontramos o botão e armazenamos na variável btn.

Na linha 3 é invocada uma função chamada addEventListener, que recebe o evento a ser observado(que queremos responder), e uma função contendo o que queremos fazer em resposta a esse evento.

A linha 4 possui o unico comando da nossa função de resposta, um alerta com uma mensagem simpática.

Percebeu que um dos parâmetros que passamos para a função addEventListener é uma função? Isso é um exemplo de função atuando como termo de primeira ordem, outra aplicação seria retornar uma função ou atribuir uma função a uma variavel.

EventListeners são o exemplo de uso de funções de primeira ordem que usaremos com mais frequência, pois são uteis para dar comportamento a aplicações web, dando a possibilidade de interação com o usuario.
